## 8.2 사용자 포털

### 로그인

1. **Google 로그인**: Google 계정으로 소셜 로그인
2. 최초 로그인 시 프로필 자동 생성
3. 이후 JWT 토큰 기반 인증 유지

### 시험 목록

1. 로그인 후 참여 가능한 시험 목록이 표시됩니다
2. 시험명, 유형, 년도, 과목 수 등의 정보를 확인할 수 있습니다
3. 원하는 시험을 선택하여 답안 입력 페이지로 이동합니다

### 답안 입력

답안 입력은 두 가지 방식을 지원합니다:

**OMR 카드 방식**

1. 실제 OMR 카드와 유사한 UI를 제공합니다
2. 각 문항의 보기 번호(①②③④⑤)를 클릭하여 답안을 선택합니다
3. 선택한 답안은 즉시 저장됩니다
4. 과목 탭을 전환하며 답안을 입력합니다

**빠른 입력 방식**

1. 숫자키(1~5)를 눌러 빠르게 답안을 입력합니다
2. Enter 키로 다음 문항으로 이동합니다
3. 대량 문항 입력 시 효율적입니다

### 답안 제출 및 채점

1. 모든 답안 입력 후 **제출** 버튼을 클릭합니다
2. 제출 즉시 자동 채점이 실행됩니다
3. 채점 결과 페이지로 자동 이동합니다

### 채점 결과 확인

채점 결과 페이지에서 다음 정보를 확인할 수 있습니다:

| 항목 | 설명 |
|------|------|
| **총점** | 전 과목 합산 점수 |
| **과목별 점수** | 각 과목의 점수 및 정답/오답 수 |
| **순위** | 전체 응시자 중 순위 |
| **표준점수** | T점수 기반 표준화 점수 |
| **합격 여부** | 합격선 및 과락 기준 판정 |
| **정오표** | 문항별 정답/오답 표시 |

### 성적 분석

1. 과목별 정답률 차트
2. 영역별 취약점 분석
3. 전체 응시자 대비 나의 위치 (백분위)
4. 과목별 표준점수 비교

### 응시 이력

1. 과거 응시한 시험 기록을 조회합니다
2. 시험별 성적, 순위, 합격 여부를 확인합니다
3. 성적 추이 그래프를 통해 학습 진행도를 파악합니다

### 프로필 관리

1. 우측 상단 프로필 아이콘을 클릭합니다
2. 표시명 등 사용자 정보를 수정할 수 있습니다

## 8.3 관리자 콘솔

### 로그인

1. **API Key 인증**: 발급된 API Key를 입력하여 로그인합니다
2. API Key는 시스템 관리자로부터 발급받습니다

### 시험 관리

**시험 등록**

1. 시험 목록 페이지에서 **시험 등록** 버튼을 클릭합니다
2. 시험 정보를 입력합니다:
   - 시험코드 (고유 식별자)
   - 시험명
   - 시험 유형 (7급, 9급 등)
   - 시험 년도
   - 합격 점수
3. **저장** 버튼으로 등록합니다

**시험 수정/삭제**

1. 시험 목록에서 해당 시험의 편집 아이콘을 클릭합니다
2. 정보를 수정한 후 저장합니다
3. 삭제 시 확인 대화상자가 표시됩니다

**시험 상태 관리**

| 상태 | 설명 |
|------|------|
| READY | 준비중 — 시험 정보 설정 단계 |
| ACTIVE | 진행중 — 응시자 답안 입력 가능 |
| COMPLETED | 완료 — 채점 완료, 결과 조회 가능 |

### 과목 관리

1. 시험 상세 페이지에서 **과목 관리** 탭을 선택합니다
2. 과목을 추가합니다:
   - 과목코드
   - 과목명
   - 문항 수
   - 문항당 배점
   - 과락 점수
3. 여러 과목을 한 번에 등록할 수 있습니다

### 정답 관리

**직접 입력**

1. 시험 상세 → **정답 관리** 탭을 선택합니다
2. 과목별로 각 문항의 정답을 입력합니다
3. 저장 버튼으로 일괄 저장합니다

**Excel 가져오기**

1. **Excel 가져오기** 메뉴를 선택합니다
2. 정답지 Excel 파일을 업로드합니다
3. **미리보기** 화면에서 데이터를 확인합니다
4. 이상 없으면 **확정** 버튼으로 등록합니다

**JSON 가져오기**

1. **JSON 가져오기** 메뉴를 선택합니다
2. 문항 데이터 JSON 파일을 업로드합니다
3. 미리보기 후 확정합니다

### 응시자 관리

**개별 등록**

1. 시험 상세 → **응시자** 탭을 선택합니다
2. **응시자 추가** 버튼을 클릭합니다
3. 수험번호, 이름, 응시지역, 답안을 입력합니다

**Excel 일괄 등록**

1. 응시자 목록 Excel 파일을 준비합니다
2. Excel 가져오기 기능으로 업로드합니다
3. 미리보기 확인 후 확정합니다

### 채점

**자동 채점**

- 응시자 답안 입력 시 자동으로 채점이 실행됩니다
- 과목별 점수, 총점, 순위가 즉시 계산됩니다

**재채점**

1. 정답이 변경된 경우 **재채점** 버튼을 클릭합니다
2. 전체 응시자에 대해 재채점이 실행됩니다
3. 순위 및 합격 여부가 재계산됩니다

### 통계 분석

**대시보드**

- 전체 시험 현황 요약
- 최근 시험 합격률 추이
- 과목별 평균 점수

**시험별 통계**

| 통계 항목 | 설명 |
|-----------|------|
| 기본 통계 | 평균, 최고점, 최저점, 표준편차, 합격률 |
| 문항 분석 | 문항별 정답률, 변별도, 난이도 |
| 취약점 분석 | 영역별/과목별 취약점 |
| 변별도 분석 | 상위/하위 그룹 정답률 비교 |
| 성적 추이 | 시험별 성적 변화 차트 |

### 문제은행

1. **문제은행** 메뉴에서 문제 그룹을 생성합니다
2. 그룹에 문항을 추가합니다 (문항 텍스트, 보기, 정답, 해설)
3. 문항은 과목 유형별로 분류됩니다

### 문제세트

1. **문제세트** 메뉴에서 새 세트를 생성합니다
2. 문제은행에서 문항을 선택하여 세트를 구성합니다
3. 문제세트를 시험에 배포할 수 있습니다

## 8.4 자주 묻는 질문 (FAQ)

### 로그인/계정

**Q: Google 로그인이 안 됩니다.**
A: 브라우저 팝업 차단을 해제하고, 쿠키를 허용해 주세요. 문제가 지속되면 시크릿 모드에서 시도해 보세요.

**Q: 관리자 API Key를 분실했습니다.**
A: 시스템 관리자에게 재발급을 요청하세요.

### 시험/채점

**Q: 답안을 수정할 수 있나요?**
A: 제출 전에는 자유롭게 수정 가능합니다. 제출 후에는 수정이 불가합니다.

**Q: 채점 결과가 맞지 않습니다.**
A: 관리자에게 정답 확인을 요청하세요. 정답이 수정되면 관리자가 재채점을 실행할 수 있습니다.

**Q: 과락이 무엇인가요?**
A: 개별 과목에서 최소 점수(과락 기준)에 미달하면, 총점이 합격선을 넘어도 불합격 처리됩니다.

### 통계

**Q: 표준점수란 무엇인가요?**
A: 원점수를 전체 응시자의 평균과 표준편차를 고려하여 표준화한 점수입니다. T점수 = 50 + 10 × (원점수 - 평균) / 표준편차로 산출됩니다.

**Q: 변별도란 무엇인가요?**
A: 해당 문항이 상위권과 하위권 수험생을 얼마나 잘 구별하는지를 나타내는 지표입니다. 상위 27% 정답률에서 하위 27% 정답률을 뺀 값입니다.

### 데이터

**Q: Excel 업로드 형식은 어떻게 되나요?**
A: 미리보기 기능으로 데이터를 확인한 후 확정할 수 있습니다. 잘못된 데이터가 있으면 미리보기에서 오류가 표시됩니다.

## 8.5 문제 발생 시

| 문제 | 해결 방법 |
|------|-----------|
| 페이지가 로딩되지 않음 | 브라우저 캐시 삭제 후 새로고침 |
| API 오류 (500) | 잠시 후 재시도, 지속 시 관리자에게 문의 |
| 로그인 세션 만료 | 재로그인 |
| Excel 업로드 실패 | 파일 형식(xlsx) 및 용량(20MB 이하) 확인 |
