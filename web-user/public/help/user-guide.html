공무원 시험 모의고사 응시 및 답안 채점 서비스

1. 서비스 소개

HopenVision은 공무원 시험 수험생을 위한 온라인 모의고사 및 자동 채점 서비스입니다. 실제 시험과 동일한 환경에서 과목별 시간 제한 모의고사를 응시하거나, 답안 번호만 빠르게 입력하여 즉시 채점 결과를 확인할 수 있습니다.

2. 주요 기능

📝

모의고사 응시

실제 시험처럼 문제를 보면서 과목별 시간 제한 내에 풀 수 있는 풀스크린 모의고사 모드입니다.

⚡

빠른 답안 채점

답안 번호만 빠르게 입력하여 즉시 채점하는 모드입니다. OMR 카드 또는 빠른 입력 방식을 지원합니다.

📊

성적 분석

과목별 점수, 정답률, 전체 응시자 대비 순위 및 백분위를 확인할 수 있습니다.

⏱

과목별 타이머

과목별 독립 타이머로 실제 시험 시간 배분을 연습할 수 있습니다. 시간 초과 시 자동으로 다음 과목으로 전환됩니다.

💾

자동 임시저장

모의고사 중 5초마다 답안이 자동 저장됩니다. 브라우저 종료 후에도 이전 응시 기록을 복원할 수 있습니다.

📆

채점 이력

이전에 응시한 시험의 채점 결과를 언제든 다시 확인할 수 있습니다.

3. 모의고사 응시 방법

  1. 시험 목록에서 [모의고사] 클릭
    채점 가능한 시험 목록에서 원하는 시험의 모의고사 버튼을 클릭합니다.
  2. 응시 과목 선택
    필수 과목(국어, 영어, 한국사)은 자동으로 선택되며, 직렬별 전문 과목 2개를 추가 선택합니다.
  3. 과목별 순차 응시
    첫 번째 과목부터 타이머가 시작됩니다. 문제를 읽고 보기 중 답을 선택한 뒤, 마지막 문제에서 과목 응시 완료를 클릭하면 다음 과목으로 이동합니다. 시간이 초과되면 자동으로 다음 과목으로 전환됩니다.
  4. 복습 모드
    5과목 모두 응시 완료 후 복습 모드에 진입합니다. 남은 시간 내에 모든 과목을 자유롭게 이동하며 답안을 수정할 수 있습니다.
  5. 답안 제출
    우측 하단의 답안 제출 버튼을 클릭하면 자동 채점이 진행되고 결과 페이지로 이동합니다.

4. 빠른 답안 채점 방법

  1. 시험 목록에서 [답안채점] 클릭
    시험 목록에서 답안채점 버튼을 클릭합니다.
  2. 응시 과목 선택
    모의고사와 동일하게 필수 과목 + 선택 과목 2개를 지정합니다.
  3. 입력 방식 선택
    OMR 카드 모드 (버튼 클릭) 또는 빠른 입력 모드 (숫자 키보드 입력) 중 선택합니다.
  4. 답안 입력 후 제출
    각 과목 탭에서 답안 번호(1~5)를 입력하고 답안 제출 및 채점 버튼을 클릭합니다.

5. 주의사항

모의고사 중 이탈 방지

모의고사 진행 중에는 풀스크린 모달이 활성화되어 다른 페이지로 이동할 수 없습니다. 시험을 중단하려면 상단의 중단 버튼을 사용하세요. 현재까지의 답안은 자동 저장되며, 다시 접속 시 복원할 수 있습니다.

자동 임시저장 및 복구

모의고사 중 답안, 타이머, 진행 상태가 5초마다 자동 저장됩니다. 브라우저가 비정상 종료되어도 24시간 이내에 재접속하면 이전 기록을 복원할 수 있습니다.

답안 제출 후 수정 불가

답안을 제출하면 즉시 채점이 진행되며, 이후 답안을 수정할 수 없습니다. 제출 전에 모든 답안을 충분히 확인하세요.

과목별 시간 제한

각 과목에는 독립된 시간 제한이 있습니다 (기본 22분). 시간이 초과되면 해당 과목은 자동으로 마감되고 다음 과목으로 이동합니다. 입력한 답안까지만 채점에 반영됩니다.

7. 자주 묻는 질문

Q. 모의고사와 답안채점의 차이점은 무엇인가요?
모의고사는 실제 시험처럼 문제를 보면서 과목별 시간 제한 내에 풀 수 있는 모드입니다. 답안채점은 이미 풀어본 시험의 답안 번호만 빠르게 입력하여 즉시 채점하는 모드입니다.
Q. 모의고사 중 브라우저를 실수로 닫았는데 복원할 수 있나요?
네, 가능합니다. 5초마다 자동 저장되므로, 같은 시험의 모의고사 페이지에 다시 접속하면 "이전 응시 기록이 있습니다" 팝업이 나타납니다. "복원"을 선택하면 답안과 타이머가 복구됩니다. 단, 저장 후 24시간이 경과한 세션은 자동 만료됩니다.
Q. 과목 시간이 초과되면 어떻게 되나요?
시간 초과 시 해당 과목은 자동으로 마감되고 다음 과목으로 전환됩니다. 마감된 과목의 답안은 입력된 상태 그대로 유지됩니다. 전체 과목 응시 완료 후 복습 모드에서는 남은 시간이 있는 과목의 답안을 수정할 수 있습니다.
Q. 선택 과목은 어떻게 정하나요?
자신의 직렬(행정, 세무, 전산 등)에 맞는 전문 과목 2개를 선택하면 됩니다. 필수 과목(국어, 영어, 한국사)은 모든 직렬 공통이며 자동 선택됩니다. 2024 국가직 9급은 필수 3과목만 등록되어 있어 별도 선택이 필요 없습니다.
Q. 한 시험에 모의고사와 답안채점을 모두 사용할 수 있나요?
아니요, 하나의 시험에 대해 한 번만 제출할 수 있습니다. 모의고사 또는 답안채점 중 하나의 방식으로 답안을 제출하면 해당 시험은 "채점완료" 상태가 되며, 이후에는 결과 조회만 가능합니다.
Q. 텍스트 모드와 이미지 모드는 무엇인가요?
모의고사 화면 상단에서 텍스트/이미지 모드를 전환할 수 있습니다. 텍스트 모드는 문제와 보기를 텍스트로 표시하며, 이미지 모드는 원본 시험지 이미지로 표시합니다. 현재 대부분의 문제는 텍스트로 등록되어 있으며, 이미지는 순차적으로 추가 예정입니다.
Q. 채점 결과에서 어떤 정보를 확인할 수 있나요?
채점 결과 페이지에서는 총점, 과목별 점수, 정답/오답 현황, 정답률, 전체 응시자 대비 추정 순위, 합격/불합격 판정, 과목별 과락 여부를 확인할 수 있습니다. 각 문항별로 본인의 답과 정답을 비교할 수도 있습니다.
