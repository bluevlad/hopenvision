## 7.1 릴리스 이력

### v0.2.0 (2025-02-06)

**설계 및 문서화**

- 구현 방안 제안서 작성
- Architecture Decision Records (ADR) 체계 구축
  - ADR-001: 시스템 아키텍처 결정 (단일 통합 앱)
  - ADR-002: 배치 시스템 설계
  - ADR-003: 차트 라이브러리 선정 (Recharts)
- WBS 전면 개편: 6개 Phase로 재구성
- 시스템을 사용자/관리자/배치로 구분
- 변경 관리 체계 (CHANGELOG) 도입

---

### v0.1.0 (2025-02-06)

**초기 릴리스**

- Spring Boot Backend 초기 구성
  - 시험 관리 API (CRUD)
  - 과목 관리 기능
  - 정답 관리 기능
  - Excel 가져오기 기능
- React Frontend 초기 구성
  - 시험 목록 페이지
  - 시험 등록/수정 페이지
  - 정답 입력 페이지
  - Excel 가져오기 페이지
- Docker Compose 설정 (PostgreSQL)
- GitHub 저장소 연결

```
MAJOR.MINOR.PATCH

PATCH: 하위 호환성 있는 버그 수정
```
